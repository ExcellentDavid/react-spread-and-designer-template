# React SpreadJS 项目模板

本模板包含了 `SpreadJS` 运行时 *Spread* 和 设计器 *Designer*，分别位于 `src/views/SpreadView.jsx` 和 `src/views/SpreadView.jsx` ，可根据项目情况酌情使用。

## 快速开始

安装依赖

```sh
npm install
```

启动开发服务器

``` sh
npm run dev
```

打包项目

```sh
npm run build
```
